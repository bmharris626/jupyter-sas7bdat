## 0.1.0

Initial release of the JupyterLab SAS7BDAT extension.

- Add a Jupyter Server extension with read and conversion endpoints under `/jupyter-sas7bdat/`.
- Add `.sas7bdat` preview support with variable metadata and paged data browsing in JupyterLab.
- Add conversion workflows for CSV, TSV, JSON, Parquet, and SAS XPORT (`.xpt`).
- Preserve SAS labels and formats in Parquet field metadata where available.
